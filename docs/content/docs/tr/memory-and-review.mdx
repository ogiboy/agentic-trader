## Operatör için hafıza ne demek

Operatör açısından hafıza, bir kararı açıklamaya yardım eden kanıttır:

- benzer context olarak getirilen önceki koşular
- market context pack'leri ve data-quality flag'leri
- specialist ve manager çıktıları
- review sırasında kullanılan broker/account durumu
- persist edilmiş trace, trade context ve review snapshot'ları
- source attribution, freshness ve missing-evidence notları

Hafıza bir inceleme desteğidir.
Gizli trade izni değildir; broker, risk veya runtime policy'yi sessizce
değiştirmemelidir.

## Review kanıtı

Şüpheli bir sonucu incelerken kanıtı şu sırayla oku:

1. `as_of`, freshness, completeness ve fallback state dahil provider/market artefact'leri
2. runtime state, supervisor state ve loglar
3. broker/account status, execution outcome ve rejection reason
4. specialist ve manager çıktıları
5. persist edilmiş review snapshot'ları ve UI davranışı

"Cevap yanlış hissettiriyor" noktasından doğrudan "prompt değişmeli" sonucuna
atlama. Önce sistemin gerçekte hangi veri, state ve kanıtı gördüğünü doğrula.

## Trading hafızası ve katkı notları

Agentic Trader'da katkı verenlere dönük repo hafızası da var. Bunlar:

- `.ai/current-state.md`
- `.ai/tasks.md`
- `.ai/decisions.md`
- `.ai/qa/*`
- mevcut çalışma varsayımlarını açıklayan docs sayfaları

Bu dosyalar bakım yapanların projeyi hizalı tutmasına yardım eder. Bunlar
operatöre gösterilen trading memory, similar-run retrieval veya review evidence
ile aynı şey değildir.

## Katkı notları ne zaman güncellenmeli

Bir değişiklik şunları etkiliyorsa `.ai` notlarını güncelle:

- runtime davranışı
- ürün yönü
- geliştirici workflow'u
- frontend ya da docs mimarisi
- QA beklentileri
- safety sınırları

## Docs sorumluluğu

Bu docs sitesi katkı verenlerin koruduğu bilgi tabanının bir parçası, ama önce
operatör için yazılmalı.
Kalıcı bir karar değiştiğinde doğru kapanış çoğu zaman şudur:

- implementasyonu güncelle
- ilgili docs sayfasını güncelle
- `.ai/current-state.md` dosyasını güncelle
- `.ai/decisions.md` dosyasını güncelle
- iş kaldıysa `.ai/tasks.md` dosyasını güncelle

Temel dil kuralı: ürün hafızasını decision/review evidence olarak, `.ai`
dosyalarını ise katkı notları olarak anlat.
